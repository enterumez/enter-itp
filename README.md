# Enter IT Partners

複雑なIT環境をシンプルに。経営視点からの情報基盤構築を支援します。

## サイト情報

- **公開URL**: https://enterumez.github.io/enter-itp/
- **ビルド方法**: `pnpm run build`
- **ホスティング**: GitHub Pages

## 更新方法

1. ローカルでサイトを修正
2. `pnpm run build` でビルド
3. `dist/public` の内容をこのリポジトリにコピー
4. `git add`, `git commit`, `git push` でGitHubにプッシュ
5. GitHub Pagesが自動的にデプロイします
